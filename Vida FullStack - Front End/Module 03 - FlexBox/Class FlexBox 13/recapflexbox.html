Module Overview

This module was simple and quick, a link with all the resources was provided beforehand, so it was just a matter of reorganizing and annotating. Not much to work with in terms of customization of each class as I've done before, as the model of this module was somewhat strict. The content was pretty simple, I only miss some more practical application, as it was a very closed circuit (a graphic, with pre-determined elements). Overall, it was more of a chill module.

Since the simpler and stricter nature of the classes, this recap will be made more simply, without really going over each class. And there's nothing more fitting than building it on top of the original graphic used on all classes.
